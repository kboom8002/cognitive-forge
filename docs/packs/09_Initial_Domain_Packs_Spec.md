# 09. Initial Domain Packs Spec

## Corporate PR Turnkey Pack
Key: pack.corporate_pr.v1. Required app: /a/corporate-pr-suite. Outputs Answer Card, Press Release, Company Profile, Web Brochure, Consistency Audit.

## Book-to-Agent Pack
Key: pack.book_to_agent.v1. Required app: /a/book-to-agent. Outputs book knowledge summary, insight QA, action plan, reflection questions.

## AI Training Practice Pack
Key: pack.ai_training_practice.v1. Required app: /a/prompt-improvement-practice. Outputs diagnosis, improved prompt, checklist, next practice.
