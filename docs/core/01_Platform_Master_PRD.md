# 01. Platform Master PRD

Cognitive Forge is a Knowledge-to-AI-App platform. It converts books, corporate knowledge, and training materials into contract-driven AI micro-apps.

## MVP Packs
1. Corporate PR Turnkey Pack: revenue wedge.
2. Book-to-Agent Pack: flagship category story.
3. AI Training Practice Pack: education adoption wedge.

## Core Jobs
- Turn static knowledge into reusable AI apps.
- Run AI tasks with input/output contracts.
- Connect CasePacks with Bridge CasePacks.
- Expose clean UI while hiding internal prompt/runtime logic.

## MVP Outcomes
- /a/corporate-pr-suite runs a graph app.
- /a/book-to-agent runs a graph app.
- /a/prompt-improvement-practice runs a casepack app.
- Trace, usage, validation, and public no-leak checks work.
